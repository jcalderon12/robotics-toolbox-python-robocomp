## Getting going

You will need Python >= 3.6

### From GitHub

To install the bleeding-edge version from GitHub

```shell script
git clone https://github.com/jcalderon12/robotics-toolbox-python-robocomp.git
cd robotics-toolbox-python
pip3 install -e .
```
